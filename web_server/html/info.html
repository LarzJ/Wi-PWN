Wi-PWN
Dit project is onder licentie van Creative Commons Naamsvermelding-NietCommercieel 4.0.

Deze software is bedoeld voor professionele gebruikers!
Ongeauthoriseerde aanvallen kunnen onbeveiligde 802.11 Wi-Fi-verbindingen verbreken, waardoor het als een Wifi Jammer beschouwd kan worden.
